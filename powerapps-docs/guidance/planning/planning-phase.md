# <a name="planning-phase"></a>計画フェーズ

計画はアプリ作成プロセスの最も重要な部分です。 計画を立てる際には、次の点を考慮する必要があります:

- アプリは **どのような問題** を解決しますか?

- **誰** がこのアプリを使用しますか?

- ユーザーにとって **どのような目標** や目的をが達成されますか?

答えを知ることは、アプリを設計するときに進捗を把握するのに役立ちます。 *問題を解決する* のではなく、*アプリを作成する* という目的に陥るのは簡単です。
