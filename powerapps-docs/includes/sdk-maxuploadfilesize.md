アップロードできるファイルの最大サイズは、**Organization.MaxUploadFileSize** プロパティによって決定されます。 このプロパティは、Dynamics 365 アプリケーションの **[システム設定]** の **[電子メール]** タブで設定します。 この設定では、メール メッセージ、メモ、および Web リソースに添付できるファイルのサイズが制限されます。 既定の設定は 5 MB です。
