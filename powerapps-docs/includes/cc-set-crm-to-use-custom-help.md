**[ヘルプ]** ボタンをクリックしたときにユーザーが正しい情報を取得できるようにするには、カスタム ヘルプを使用するように [!INCLUDE[pn_crm_shortest](pn-crm-shortest.md)] を設定する必要があります。 **[設定]**  >  **[管理]** に移動し、 **[システム設定]** をクリックし、 **[全般]** タブをクリックし、 **[カスタマイズ可能なエンティティにカスタム ヘルプを使用]** に **[はい]** を選択します。
