`getTimeZoneOffsetMinutes(date)`

## <a name="parameters"></a>パラメーター

| パラメーター名|型|必須出席者|説明|
| ------------- |----|--------|-----------|
|日付|`Date`|はい|UTC からのオフセットを取得する日付。|

## <a name="return-value"></a>戻り値

種類: `number`
