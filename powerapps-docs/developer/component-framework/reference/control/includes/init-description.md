コンポーネントインスタンスを初期化するために使用します。 コンポーネントは、リモートサーバー呼び出しやその他の初期化操作を開始できます。 データセットの値をここで初期化することはできません。 [Updateview](../updateview.md)メソッドを使用して設定してください。
