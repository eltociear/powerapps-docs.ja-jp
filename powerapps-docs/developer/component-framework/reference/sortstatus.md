## <a name="properties"></a>プロパティ

## <a name="name"></a>名前

列の名前

**種類**: `string`

## <a name="sortdirection"></a>sortDirection

<!-- ColumnSortDirection  -->
現在の列の並べ替え方向。

**種類**: `enum`

`sortDirection` 値は以下の可能な値を含む列挙値です

|Value|メンバー|
|--|--|
|-1|なし|
|0|昇順|
|1|降順|
