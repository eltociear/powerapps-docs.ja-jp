# <a name="keyboard-shortcuts-for-canvas-apps"></a>キャンバスアプリのキーボードショートカット

> [!NOTE]
> ショートカットは、キーボードレイアウトによって異なる場合があります。

## <a name="file"></a>ファイル

| カット | 操作 |
|--|--|
| Ctrl + O (または Alt + F) | ファイルを開きます。 |
| Ctrl + Shift + S (または Alt + P) | アプリを別の名前で保存します。 |
| Ctrl + S | アプリを同じ名前で、または初めて保存します。 |
| Ctrl + Shift + P | アプリを保存し、[発行] ダイアログをアクティブにします。 |
| F12 | アプリファイル (msapp) をダウンロードします。 |
| Alt + F | **[ファイル]** メニューを開きます。 |

## <a name="ribbon"></a>リボン

| カット | 操作 |
|--|--|
| 以下に | 選択したコマンドを実行します。 |
| タブ | 選択したタブのコマンド間を移動し、次のタブに移動します。 |
| Ctrl + F6 | 次のランドマークに移動します。 |
| Ctrl + Shift + F6 | 前のランドマークに移動します。 |
| Alt + I | **[挿入]** タブを選択します。 |

## <a name="editing"></a>編集中

| カット | 操作 |
|--|--|
| Ctrl&lt;/localizedText&gt; + &lt;localizedText&gt;A | [すべて] を選択します。 |
| Ctrl+X | 閉. |
| Ctrl+C | [コピー]: |
| Ctrl+V | [貼り付け]: |
| Ctrl+Z | Undo コマンド。 |
| Ctrl+Y | やり直しコマンド。 |
| Ctrl + M | 画面を追加します。 |
| Ctrl + = または Ctrl + Shift + = | 拡大します。 |
| Ctrl +-または Ctrl + Shift +- | 縮小します。 |
| Ctrl + 0 | キャンバスをページに合わせる。 |
| Shift + Enter | 数式内の行を分割します。 |

## <a name="preview"></a>プレビュー

| カット | 操作 |
|--|--|
| F5 | プレビューモードを開始します。 |
| Esc | プレビューモード、ダイアログボックス、またはポップアップウィンドウを閉じます。|

## <a name="canvas"></a>Canvas

| カット | 操作 |
|--|--|
| タブ | 次のコントロールを選択します。 |
| Ctrl キーを押しながらクリックするか Shift キーを押しながらクリック | 一度に複数のオブジェクトを選択します。 |
| 右向きの矢印 | 選択したコントロールを右に微調整します。 |
| 左方向キー | 選択したコントロールを左に微調整します。 |
| 上矢印 | 選択したコントロールを上に移動します。 |
| 下向きの矢印 | 選択したコントロールを下に移動します。 |

## <a name="tree-view"></a>ツリー構造

> [!NOTE]
> これらのショートカットでは、**ツリービュー**ペインにフォーカスを設定する必要があります。

| カット | 操作 |
|--|--|
| F2 | コントロールの名前を変更します。 |
| Esc | コントロールの名前変更をキャンセルします。 |
| Ctrl + G | コントロールをグループ化またはグループ解除します。 |
| Ctrl + 終わり角かっこ (]) | コントロールを前面へ移動します。 |
| Ctrl + 始め角かっこ ([) | コントロールを後方に送信します。 |
| Ctrl + Shift +] | 前面へ移動します。 |
| Ctrl + Shift + [ | 最背面へ移動します。 |

## <a name="resize"></a>サイズ変更

| カット | 操作 |
|--|--|
| Shift + ← | 幅を狭くします。 |
| Ctrl + Shift + ← | 幅を少し狭くします。 |
| Shift + ↓ | 高さを小さくします。 |
| Ctrl + Shift + ↓ | 高さを少し小さくします。 |
| Shift + → | 幅を拡大します。 |
| Ctrl + Shift + → | 幅を少し大きくします。 |
| Shift + ↑ | 高さを拡大します。 |
| Ctrl + Shift + ↑ | 高さを少し大きくします。 |

## <a name="text-format"></a>テキスト形式

| カット | 操作 |
|--|--|
| Ctrl + B  | 太字のレベルを切り替えます。 |
| Ctrl + I | 斜体をオンまたはオフにします。 |
| Ctrl + U | 下線を追加または削除します。 |

## <a name="alternate-behavior"></a>代替動作

| カット | 操作 |
|--|--|
| Alt または Ctrl + Shift | 1. コントロールを選択する前に、アプリのユーザーと同じようにコントロールを操作できるように、デザイン要素を非表示にします。<br>2. コントロールのサイズ変更または位置変更を開始した後、これらのキーを押すとスナップポイントが上書きされます。 |

Excel スプレッドシートと同様に、キャンバスアプリは編集中でもライブで動作します。  アプリを実行するためにプレビューモードに変更する必要はありません。編集とテストのサイクルが大幅に短縮されます。  しかし、これには問題があります。ボタンコントロールを選択して、アプリケーションのロジックを実行するようにボタンコントロールを選択してサイズを変更できるようにするには、どうすればよいでしょうか。

デザインモードの場合、既定では、オブジェクトを選択するときに、移動、サイズ変更、プロパティの変更、およびオブジェクトの構成が行われます。  この既定値をオーバーライドするには、Alt キーまたは Ctrl キーを押しながら Shift キーを押して選択*を開始し*てから、アプリのユーザーがその選択を処理したと見なします。  

次のアニメーションでは、最初にボタンコントロールが編集用に選択されています。  装飾はコントロールの周囲に表示され、数式バーは**Onselect**プロパティを表示して編集する準備ができています。  ボタンが解放されます。  *Alt キーを押すと*、ボタンコントロールがもう一度選択されますが、このとき、 **onselect**プロパティが評価され、実行中のアプリでボタンが選択された場合と同じように、通知が表示されます。  

![Alt キーを押しながらボタンコントロールを選択することによる効果を示すアニメーション](media/keyboard-shortcuts/alt-select.gif)

Alt キーは、移動とサイズ変更のためにスナップポイントを上書きするコントロールを選択し*た後*にも使用できます。  次のアニメーションは、[**編集フォーム**](controls/control-form-detail.md)コントロール内のデータカードのサイズ変更を示しています。  初期状態では、サイズ変更は特定のスナップポイントに限定されます。  その後、 *mouses ボタンを離さず*に、マウスボタンに加えて Alt キーが押されます。 Alt キーを追加すると、スナップポイントが上書きされ、任意の幅をマウスで取得できます。 

![Alt キーをデータカードのサイズ変更に追加した場合の効果を示すアニメーション](media/keyboard-shortcuts/alt-fine-control.gif)
