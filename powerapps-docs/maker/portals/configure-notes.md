# <a name="configure-notes-for-entity-forms-and-web-forms-on-portals"></a>ポータルでエンティティフォームと web フォームのメモを構成する

Subgrids の場合と同様に、ポータルでマネージフォームにメモを追加するのは&mdash;簡単です。[フォームデザイナー](../model-driven-apps/create-design-forms.md)を使用して、モデルドライブのアプリフォームにメモコントロールを追加するだけでかまいません。これで完了です。 メタデータを使用して、メモコントロールの動作を構成できます。

> [!Note]                                                           
> すべてのメモがポータルに表示されるには、明示的な[エンティティアクセス許可](configure/assign-entity-permissions.md)が必要です。 読み取りと編集を行うには、読み取りと書き込みの権限が付与されている必要があります。 作成には、2つのアクセス許可が存在する必要があります。 "作成" および "追加" の特権を持つアクセス許可をメモ (注釈) エンティティに付与する必要があります。2番目のアクセス許可は、追加先の特権を付与して、メモがアタッチされているエンティティ型に割り当てる必要があります。 エンティティのアクセス許可を有効にするには、対応するエンティティフォームまたは web フォームの手順で [**エンティティのアクセス許可を有効**にする] チェックボックスをオンにする必要があります。

## <a name="notes-configuration-for-entity-forms"></a>エンティティフォームのメモ構成

1. [ポータル管理アプリ](configure/configure-portal.md)を開きます。
2. **ポータル** > **コンテンツ** > **エンティティフォーム**にアクセスします。 アクティブなエンティティフォームの一覧が表示されます。
3. メモの構成を追加するエンティティフォームを選択します。
4. エンティティフォームの**メタデータ**に移動するには、上部のドロップダウンリストを使用するか、使用しているエンティティフォームレコードのメインフォームのサブグリッドを使用します。
5. 新しいレコードを追加するには、 **[新しいエンティティフォームメタデータの追加]** を選択します。
6. **[種類]** ドロップダウンリストから、 **[メモ]** を選択します。 メモの構成&ndash;特定の設定が表示されます。 既定では、ほとんどの設定が折りたたまれています。 セクションを展開して、追加の設定を表示できます。
7. フィールドに適切な値を入力して入力します。 [属性](#attributes)の [!include[](../../includes/proc-more-information.md)]、[ダイアログ](#create-dialog-options)オプションの作成、ダイアログ[オプションの編集](#edit-dialog-options)、および[ダイアログオプションの削除](#delete-dialog-options)
8. フォームを保存します。

    ![エンティティフォームのメモ構成を追加する](media/add-note-configuration.png "エンティティフォームのメモ構成を追加する")  

    構成を追加すると、ポータルで有効になっている適切なオプションを使用して、ノートコントロールがレンダリングされます。

### <a name="assign-entity-permissions"></a>エンティティのアクセス許可を割り当てる

次のようにして、適切なエンティティのアクセス許可を作成し、レコードに割り当てる必要があります。そうしないと、メモの **[追加]** 、 **[編集]** 、および **[削除]** の各ボタンが非表示になります。

- スコープを**グローバル**として使用して、**アクティビティ (activitypointer)** エンティティの読み取り、書き込み、作成、追加、および追加の特権を付与します。 このエンティティアクセス許可は、ユーザーの web ロールに関連付けられている必要があります。
- メモコントロールが有効になっているエンティティの読み取り、書き込み、作成、追加、および追加の特権。 スコープは**グローバル**に設定する必要があります。 このエンティティアクセス許可は、ユーザーの web ロールに関連付けられている必要があります。

    ![エンティティのアクセス許可の追加](media/entity-permission.png "エンティティのアクセス許可の追加")

    ![エンティティアクセス許可への web ロールの追加](media/entity-permission-web-roles.png "エンティティアクセス許可への web ロールの追加")

カスタムフォームを作成し、そこにメモセクションを追加した場合は、表示する既定のタブとして **[メモ]** を選択してください。

![カスタムフォームのメモ](media/notes-activities-tab.png "カスタムフォームのメモ")

## <a name="notes-configuration-for-web-forms"></a>Web フォームのメモ構成

Web フォームメモは、[エンティティフォームメモ](#notes-configuration-for-entity-forms)と同じように構成されます。 まず、メモが含まれている Web フォームの手順のメタデータレコードを作成してから、メモ構成メタデータを追加する必要があります。
