# <a name="configure-web-form-metadata-for-portals"></a>ポータルの web フォームメタデータの構成

Web フォームメタデータには、ネイティブのエンティティフォーム編集機能では不可能な、フォームフィールドの機能を拡張またはオーバーライドするための追加の動作変更ロジックが含まれています。

## <a name="add-a-new-record"></a>新しいレコードを追加する

1. 変更するフィールドが含まれている Web フォームの手順で、**関連**する > **メタデータ**にアクセスします。 

2. **[新しい Web フォームメタデータ]** を選択します。

## <a name="web-form-metadata-properties"></a>Web フォームメタデータのプロパティ

次の属性は、フォーム上の要素に対して追加のスタイル設定と機能を提供します。

| 名前          | Description                                                                                                                                                                                                                                                                                                                                          |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Web フォームのステップ | Web フォームのメタデータレコードに関連付けられている Web フォームのステップ。                                                                                                                                                                                                                                                                                      |
| 種類          | 使用可能なオプションは次のとおりです。<ul><li>属性</li><li>セクション</li><li>タブ</li></ul>型の値として属性を選択すると、関連するステップに対してレンダリングされた現在のフォームのフィールドを変更するための適切なオプションが表示されます。 [種類] の値として [セクション] を選択すると、フォーム上のセクションを変更するために使用できるオプションが表示されます。 型の値として [Tab] を選択すると、フォームのタブを変更するために使用できるオプションが表示されます。  |

## <a name="web-form-metadata-type--attribute"></a>Web フォームメタデータ型 = 属性

選択された型が ' Attribute ' の場合、次のプロパティが表示されます。


|          名前          |                                                                                                                                                    Description                                                                                                                                                     |
|------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 属性の論理名 |                                                                                                                              変更する属性フィールドの論理名。                                                                                                                               |
|         ラベル          | エンティティの属性に割り当てられた既定のラベルを、この入力で指定されたテキストに置き換えます。 Common Data Service 環境に対してインストールされ、有効になっている各言語パックについて、関連付けられた言語でメッセージを入力するためのフィールドを使用できるようになります。 |

### <a name="validation"></a>検証

次のセクションには、さまざまな検証パラメーターとエラーメッセージを変更するプロパティが含まれています。

Common Data Service 環境にインストールされ、有効になっている各言語パックについて、関連付けられた言語でメッセージを入力するためのフィールドを使用できるようになります。

| 名前                                        | Description                                                                                                                                                                                                                                                      |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 検証エラーメッセージ                    | フィールドの既定の検証エラーメッセージをオーバーライドします。                                                                                                                                                                                                    |
| 正規表現                          | フィールドを検証するために追加される正規表現。                                                                                                                                                                                                          |
| 正規表現の検証エラーメッセージ | 検証された正規表現が失敗した場合に表示される検証エラーメッセージ。                                                                                                                                                                               |
| フィールドが必要です                           | 属性フィールドに値を含める必要があることを確認します。                                                                                                                                                                                                   |
| 必須フィールドの検証エラーメッセージ     | フィールドに値が含まれていない場合、既定の必須フィールドエラーメッセージを上書きします。                                                                                                                                                                        |
| 範囲の検証エラーメッセージ              | フィールドの値が、整数型、10進数、浮動小数点数、または通貨型のエンティティ属性で指定された適切な最小値と最大値の範囲外の場合に表示される、既定の範囲検証エラーメッセージをオーバーライドします。 |
| 位置情報検証コントロールのエラーメッセージ         | 属性が単一行のテキストで、指定されたコントロールスタイルが位置情報参照検証ツールとして1行のテキストである場合、これは入力の検証が失敗した場合に表示される既定のエラーメッセージを上書きします。 |
| 定数の合計検証エラーメッセージ       | 属性が整数型で、指定されたコントロールスタイルが定数合計としてグループの整数である場合に適用されます。これは、入力の検証が失敗した場合に表示される既定のエラーメッセージを上書きします。                    |
| 複数選択の検証エラーメッセージ    | 属性が2つのオプションの種類であり、指定されたコントロールスタイルが複数選択されている場合に適用されます。これは、入力の検証が失敗した場合に表示される既定のエラーメッセージを上書きします。                                         |
| 順位付けの順序に誤りのある検証エラーメッセージ | 属性が整数型であり、指定されたコントロールスタイルが順位付けされたグループの整数である場合、適用されます。これにより、入力の検証が失敗した場合に表示される既定のエラーメッセージがオーバーライドされます。              |
