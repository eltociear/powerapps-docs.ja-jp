# <a name="use-keyboard-shortcuts-in-power-apps"></a>Power Apps でキーボード ショートカットを使用する

キーボード ショートカットは、一般的なタスクを実行するための、マウスの使用や指でのタップに代わる方法です。 インターフェイスをシームレスに移動するのに役立ちます。 次のキーボード ショートカットは、アプリ全体で使用できます。 

> [!NOTE]
> ここで説明するキーボード ショートカットは、米国のキーボードのレイアウトに基づいています。 他のキーボード レイアウトのキーは、米国のキーボードのキーと正確に対応していない場合があります。

## <a name="navigating-submenus"></a>サブメニュー間を移動する
スクリーン リーダーを使用する場合は、次の手順に従うと、ナビゲーション リンク上にマウス ポインターを置いたときに表示されるすべてのサブメニューにアクセスできます。

1. 標準リンク ナビゲーションを使用してメニュー オプション間を移動します。
2. 通常どおり Enter キーまたは Space キーを選択して、リンクをアクティブにします。 サブメニューの一覧が表示されます。
3. これらのサブメニュー オプションにアクセスするには、支援技術 (AT) ショートカットキーを使用して次のボタンに移動します。 たとえば、NVDA スクリーン リーダーを使用する場合は、"b" ショートカットキー (NVDA + b) を使用します。 すべてのオプション間を移動するには、キーを繰り返し選択します。

> [!IMPORTANT]
> ユーザー インターフェイス (UI) のいくつかの領域には、メニュー用の自動折りたたみ機能が組み込まれています。 これには、ナビゲーション バーとフォーム ナビゲーションが含まれます。 どちらも選択したときに展開されますが、非アクティブな状態が 30 秒経過すると項目は折りたたまれます。

## <a name="navigating-through-stages-of-a-process"></a>プロセスのステージ間を移動する
このセクションでは、たとえば潜在顧客などのレコードを表示している場合、営業プロセス内のさまざまなステージ間を移動して潜在顧客を限定するためのキーボード ショートカット、およびそれらのステージでの選択方法を示します。 たとえば、[見込みあり] ステージから [クローズ] ステージに簡単に移動するには、次のキーボード操作を試してください。
- ステージを選択したら、下 (↓) 方向キーを押してフィールドを表示します。
- ステージが折りたたまれている場合は、Enter を押して展開します。

  > [!NOTE]
  > プロセス名は、ページの上部ではなく下部にあるアイコンです。

- **Windows のキーボードショートカット**: Windows オペレーティング システムを使用している場合、次のリンクを選択し、ドロップダウン リストからバージョン番号を選択します。 Windows 環境で使用可能なすべてのショートカットを表示できます。 [Windows のキーボード ショートカット](https://support.microsoft.com/help/12445/windows-keyboard-shortcuts)に関するページを参照してください。 
- **Mac のキーボード ショートカット**: Mac オペレーティング システムを使用している場合は、次のリンクを選択し、Mac 環境で使用可能なすべてのショートカットを表示します。 「[Mac のキーボードショートカット](https://support.apple.com/HT201236)」を参照してください。
